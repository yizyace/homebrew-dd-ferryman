# homebrew-dd-ferryman

Homebrew tap for [dd-ferryman](https://github.com/yizyace/dd_ferryman) â€” local DNS + HTTPS for `.test` domains on macOS.

## Install

```bash
brew tap yizyace/dd-ferryman
brew install dd-ferryman
```

## Usage

```bash
# Start manually
sudo dd-ferryman start

# Or install as a service (starts at boot)
sudo dd-ferryman install
```
